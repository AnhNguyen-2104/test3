User Guide
Hướng dẫn sử dụng DACDT 2026

1. Dashboard

Hiển thị trạng thái kết nối PLC, vị trí 3 trục (X, Y, Z), tốc độ, error code, và Program Monitor theo dõi dòng lệnh đang chạy.

  • CONNECT PLC Q — Kết nối PLC Mitsubishi qua MX Component (logical station)
  • HOME — Về gốc máy (giữ nút)
  • RESET — Reset lỗi trục (giữ nút)
  • START ACTION — Bắt đầu chạy chương trình đã nạp (giữ nút)
  • EMERGENCY STOP — Dừng khẩn cấp
  • Manual Jog — Điều khiển tay bằng nút ←↑→↓ Z+/Z-

2. DXF/GCODE RUN

Mở file DXF hoặc G-code, xem bản vẽ 2D, chỉnh sửa G-code, và gửi xuống PLC.

  • Open — Mở file DXF (.dxf) hoặc G-code (.nc, .gcode, .g)
  • New — Tạo file G-code mới (trống)
  • Save G-code — Lưu file G-code (chọn đường dẫn lần đầu)
  • Send CAD to PLC — Gửi toàn bộ chương trình xuống buffer PLC
  • Clear Buffer — Xóa buffer PLC trước khi nạp chương trình mới

Geometry Data — Bảng tọa độ các đoạn Line/Arc/Circle từ file

Process Table — Bảng lệnh chạy gửi xuống PLC (Motion type, M code, Speed, tọa độ đích)

3. G-code hỗ trợ

G00Di chuyển nhanh (rapid) — dùng tốc độ từ Settings
G01Nội suy tuyến tính — tốc độ từ F
G02Nội suy cung tròn CW (thuận kim) — I/J hoặc R
G03Nội suy cung tròn CCW (ngược kim) — I/J hoặc R
G54–G59Chọn hệ tọa độ phôi (offset cộng vào tọa độ)
G90Chế độ tọa độ tuyệt đối
G91Chế độ tọa độ tương đối
M03Bắt đầu dispensing (bật keo)
M04Kết thúc dispensing (tắt keo)

Quy tắc modal: Dòng chỉ có tọa độ (không có G) sẽ dùng lệnh G trước đó. Dòng chỉ có X hoặc Y sẽ giữ nguyên trục còn lại.

4. DXF hỗ trợ

Đọc được: Line, Polyline2D, Polyline3D, Arc, Circle, Block/Insert

Tự động gán M03 (đầu path) và M04 (cuối path) cho mỗi quỹ đạo.

5. Settings

  • G00 Rapid Speed — Tốc độ G00 (mm/min), áp dụng cho tất cả lệnh G00
  • Workspace Size — Kích thước vùng làm việc hiển thị trên bản vẽ 2D
  • DXF Offset & Speed — Offset X/Y, Speed, Dwell M03/M04 cho file DXF
  • G54–G59 WCS Offsets — Offset tọa độ cho từng hệ tọa độ phôi (G-code)

Tất cả settings được lưu vào app_settings.txt và tự động load khi mở app.

6. Quy trình vận hành

  1. Kết nối PLC (nhấn CONNECT PLC Q)
  2. Mở file DXF hoặc G-code (tab DXF/GCODE RUN → Open)
  3. Kiểm tra Process Table và bản vẽ 2D
  4. Cài đặt Speed, Offset nếu cần (tab Settings)
  5. Nhấn Clear Buffer → Send CAD to PLC
  6. Nhấn giữ START ACTION để chạy
  7. Theo dõi Program Monitor trên Dashboard
