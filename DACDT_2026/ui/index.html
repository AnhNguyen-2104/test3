User Guide
DACDT 2026 — Gantry Robot Control System

1. Dashboard

Displays PLC connection status, 3-axis positions (X, Y, Z), speed, error codes, and Program Monitor tracking the current running line.

  • CONNECT PLC Q — Connect to Mitsubishi PLC via MX Component (logical station)
  • HOME — Return to machine home (hold button)
  • RESET — Reset axis error (hold button)
  • START ACTION — Start running the loaded program (hold button)
  • EMERGENCY STOP — Emergency stop all axes
  • Manual Jog — Manual control using ←↑→↓ Z+/Z- buttons

2. DXF/GCODE RUN

Open DXF or G-code files, view 2D drawing, edit G-code, and send to PLC.

  • Open — Open DXF (.dxf) or G-code (.nc, .gcode, .g) file
  • New — Create new empty G-code file
  • Save G-code — Save G-code file (choose path on first save)
  • Send CAD to PLC — Send entire program to PLC buffer memory
  • Clear Buffer — Clear PLC buffer before loading new program

Geometry Data — Coordinate table of Line/Arc/Circle segments from file

Process Table — Command table sent to PLC (Motion type, M code, Speed, end coordinates)

3. Supported G-code

G00Rapid move — uses speed from Settings
G01Linear interpolation — speed from F parameter
G02Circular interpolation CW (clockwise) — I/J or R
G03Circular interpolation CCW (counter-clockwise) — I/J or R
G54–G59Select work coordinate system (offset added to coordinates)
G90Absolute coordinate mode
M03Start dispensing (glue ON)
M04Stop dispensing (glue OFF)

Coordinate Systems

  • Machine Home — Fixed reference point on the machine, defined by manufacturer.
  • Work Coordinate (G54–G59) — Origin point (X=0, Y=0) set by operator on the workpiece. Offset configured in Settings.

Modal rules: A line with only coordinates (no G command) uses the previous G command. A line with only X or Y keeps the other axis unchanged.

4. Supported DXF Entities

Supported: Line, Polyline2D, Polyline3D, Arc, Circle, Block/Insert

Automatically assigns M03 (path start) and M04 (path end) for each trajectory.

5. Settings

  • G00 Rapid Speed — Speed for all G00 commands (mm/min)
  • Workspace Size — Work area dimensions shown on 2D drawing
  • DXF Offset & Speed — Offset X/Y, Speed, Dwell M03/M04 for DXF files
  • G54–G59 WCS Offsets — Coordinate offsets for each work coordinate system (G-code)

All settings are saved to app_settings.txt and auto-loaded on startup.

6. Operating Procedure

  1. Connect PLC (press CONNECT PLC Q)
  2. Open DXF or G-code file (DXF/GCODE RUN tab → Open)
  3. Verify Process Table and 2D drawing
  4. Configure Speed, Offset if needed (Settings tab)
  5. Press Clear Buffer → Send CAD to PLC
  6. Hold START ACTION to run
  7. Monitor progress on Dashboard (Program Monitor)

7. Controls & Shortcuts

  • Mouse wheel on 2D drawing — Zoom in/out
  • Mouse drag on 2D drawing — Pan (move view)
  • EXIT APP (sidebar) — Close application
